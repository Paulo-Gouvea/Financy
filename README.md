Para executar o projeto:
-> Clone o repositorio
-> Entre na pasta de Backend
-> Preencha as variaveis ambiente 
-> execute npm install
-> execute npx prisma generate
-> execute npm run dev
-> Entre na pagina de frontend
-> Preencha as variaveis ambiente
-> execute npm install
-> execute npm run dev
